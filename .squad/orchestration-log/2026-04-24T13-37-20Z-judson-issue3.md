## Problem
Streamlit executes multipage page scripts from their script directory, which can omit the repository root from Python's import search path. Absolute imports like `from app.models.connection import ADMEConnection` were failing in `app/pages/1_⚙️_Settings.py`.

## Solution
Minimal, idempotent bootstrap at the top of both `app/main.py` and `app/pages/1_⚙️_Settings.py`:
```python
import sys
from pathlib import Path
repo_root = Path(__file__).parent.parent
if str(repo_root) not in sys.path:
    sys.path.insert(0, str(repo_root))
```
